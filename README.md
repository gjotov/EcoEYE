**EcoEYE** — это интеллектуальная система экологического мониторинга города, созданная на базе компьютерного зрения (YOLOv8), спутниковых данных (Sentinel-5P) и Telegram-ботов. Проект анализирует городские камеры, оценивает трафик, вычисляет выбросы углекислого газа (CO2) и отслеживает промышленные выбросы в атмосферу.

> ⚠️ **Важный дисклеймер:**
> Это не официальный пост наблюдения и не сертифицированная лаборатория. Это пет-проект, написанный энтузиастом. Данные системы не обладают точностью профессиональных газоанализаторов. ИИ может ошибаться (принимать туман за дым и т.д.). Проект является цифровым экспериментом по применению современных технологий для мониторинга экологии родного города (г. Волжский). 
> *Источником видеопотока служат открытые городские камеры провайдера [Powernet](https://cam.powernet.com.ru).*

---

## Что умеет система

* **Анализ трафика и расчет CO2:** Распознавание автомобилей (YOLOv8) на видеопотоке с городских камер в реальном времени. Классификация транспорта (легковой/тяжелый) и математический расчет примерного объема выбросов CO2 в атмосферу.
* **Детекция промышленных выбросов:** Специально обученная нейросеть (100 эпох, mAP ~0.80) для анализа промзон. ИИ натренирован отличать реальный дым от пара, облаков или ночного освещения.
* **Спутниковый анализ:** Интеграция данных со спутника **Sentinel-5P** для построения тепловых карт загрязнения диоксидом азота (NO2) над городом и поймой.
* **Telegram-интеграция:** Автоматическая генерация отчетов. Бот присылает кадры с камер, статистику по машинам, уровень пробок и экологический статус прямо в Telegram-канал.
* **Высокопроизводительный стриминг:** Захват видеопотока осуществляется с помощью кастомного микросервиса, написанного на **Go**.

## Структура проекта

* `main.py` — Главный модуль, оркестратор всей системы и генератор отчетов.
* `vision.py` — Модуль компьютерного зрения (инференс YOLO для машин и дыма).
* `config.py` — Файл конфигурации (ссылки на RTSP-потоки, токены Telegram, параметры Sahi).
* `streamer/` — Go-сервис для стабильного захвата IP-камер.
* `modules/` — Вспомогательные скрипты (расчет математики CO2, работа со спутниковыми API).
* `yolov8l.pt` — Веса базовой модели (для трафика) + (кастомные веса для дыма).
* `basemap.png` — Базовая карта-подложка для визуализации геоданных.
* `start.bat` — Скрипт для мгновенного запуска комплекса на Windows.
